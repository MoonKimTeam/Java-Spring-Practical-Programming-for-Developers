# ch.11 알아두면 유용한 스프링 활용법

## 타입 기반 주입

- 스프링에서는 @Autowired 사용해서 의존성 주입
- 인터페이스처럼 추상 타입이고 구현체가 여러 개면 @Qualifier, @Primary 사용하여 특정 빈 지정 가능
- List 타입이면 제네릭 타입과 일치하는 모든 스프링 빈을 찾아 List의 요소로 넣어주는 방식으로 처리
- 개발 시에 확장 가능한 형태로 코드를 작성한다면 switch문 같이 확장이 어려운 코드도 쉽게 처리 가능

## 자가 호출

- 자가 호출은 굉장히 빈번하게 일어나지만, 스프링의 빈 메서드에서 발생하면 문제 발생
- 서비스 내에서 자기 호출 발생하면 호출되는 메서드에 적용된 AOP 애너테이션이 동작하지 않기 때문
